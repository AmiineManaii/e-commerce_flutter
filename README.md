# 🎮 GameMart

**GameMart** est une application e-commerce hybride développée avec **Flutter**, dédiée à l'achat et à la consultation de jeux vidéo.

## 🌟 Fonctionnalités Principales
- 🛍️ Catalogue de jeux complet avec filtrage et recherche.
- 👤 Gestion des profils utilisateurs et des adresses.
- 🛒 Panier en temps réel et processus de commande.
- ⭐ Système d'avis et de notes pour chaque jeu.
- 🎥 Visualisation des trailers YouTube directement dans l'app.

## 🛠️ Stack Technique
- **Frontend** : Flutter (Dart)
- **Backend** : JSON Server
- **Hébergement API** : Render
- **URL API** : [https://e-commerce-3qt8.onrender.com](https://e-commerce-3qt8.onrender.com)

## 🚀 Installation & Lancement
1. Clonez le repository.
2. Assurez-vous d'avoir Flutter installé sur votre machine.
3. Exécutez `flutter pub get` pour installer les dépendances.
4. Lancez l'application avec `flutter run`.
